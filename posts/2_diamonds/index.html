2. Web App: Diamond Price Recommendation

This project aimed to create an app that gives price recommendations for diamonds. You can use and test the app by accessing the following link:

Furthermore, a Jupyter notebook is available on my Kaggle profile to see a step-by-step, detailed data analysis and check how the ML models were built and compared to each other:

Objectives

  • Get insights from the data by an explanatory data analysis (EDA).
  • Build a machine learning model capable of recommending diamond prices.
  • Deploy the model into a web app.

Methodology

  • Get data from an open data source (Kaggle).
  • Perform EDA to better understand the data.
  • Data preprocessing, which included dealing with outliers, checking distributions and applying data transformations, scaling data, etc.
  • Train and test several regression models and pick the best performing.
  • Create data pipelines and develop the web front-end.
  • Deploy the ML model using Streamlit Cloud services.

Results

Five different models were trained: linear regression, lasso, ridge, polynomial, and catboost. The use of GridSearchCV helped to tune the hyperparameters. Cross-validation with five-fold was used to test every model and measure its performance. The figure below shows the final results, where Catboost regressor was the best-performing of all.
