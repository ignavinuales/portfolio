2. Face mask detection (Computer Vision)

This project aimed to retrain a YOLOv7 model to detect people wearing face masks. Moreover, I created and deployed a web app on Streamlit for users to test the model.

Feel free to try ou the app:

Refer to the GitHub repo to check the code for the project:

Objectives

The objective of this project is to use the official pre-trained YOLOv7 model (built on PyTorch) and retrain it on a custom dataset to detect people wearing face masks. The class labels are:

  • Wearing mask
  • Not wearing mask
  • Worn incorrectly

Methodology

  1. First, some of the data were collected from open data sources. Then, I collected some other data by gathering personal photos from 2020 to 2022.
  2. The unlabeled data were manually labelled using roboflow.
  3. After researching open-source object detection algorithms, I picked Yolov7. You can access the published paper here.
  4. I trained the Yolov7 model on my custom data.
  5. I measured the model performance and collect new data to improve accuracy/recall. Example: the model was not performing well on incorrectly worn masks. Therefore, I collected and labelled new data from that class and retrained the model.
  6. After obtaining a well-performing model, the deployment was done on Streamlit Cloud services for user testing.

Results

The model obtained a mean average precision (mAP) of 0.761 on the held-out test dataset. The mAP of each class was:

  • With mask: 0.944
  • Without mask: 0.836
  • Mask worn incorrectly: 0.502.

So, the global mAP is pulled down by the class mask worn incorrectly. The precision-recall curve is shown below:

Furthermore, the test data includes many complex images with big crowds of people (like the one below), making it hard for the algorithm to detect. Depending on the use case of the model, the precision could be much higher. If it detects only a few people at a time, it would perform better.

Conclusion and further work

The model performs well in less crowded places. It could easily be deployed on a building entrance to check if people entering are wearing masks and trigger an alert if they are not.

Moreover, the model should be retrained using even more data of people wearing masks incorrectly, as the mAP for such class was still low on the test data.
